## 📌 What is OverTheWire Bandit?

**Bandit** is a beginner-friendly wargame created by **OverTheWire**.  
It is designed to help beginners learn important Linux and security concepts by solving practical challenges.

Each level gives you a task, and the goal is to find the password for the next level.

Through Bandit, you can practice topics such as:

- SSH connection
- Linux commands
- File listing
- Reading files
- Hidden files
- File permissions
- Searching files
- Basic command-line usage
- Problem-solving in a Linux environment
